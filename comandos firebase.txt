Una de las opciones gratuitas de hosting es Firebase de Google
tiene sus tools y comandos de npm mediante la instalacion de un paquete global para
poder usar sus comandos para autenticar tu cuenta, tu proyecto y activar el hosting.

Nota: a tu proyecto me refiero a que debes de ir a la consola de firebase (la pagina oficial de firebase)
y crear un proyecto donde seleccionaras el hosting y el tipo de proyecto.
Firebase te guiara en la creacion y configuracion del mismo.

1. Primero se inicia por instalar el paquete con los comandos de firebase:
    - npm install -g firebase-tools
2. para verificar la version (y comprobar si esta instalado):
    - firebase --version

3. despues de comprobar que ya esta instalado y funcionando, necesitas que firebase autentifique la cuenta
    se estara utilizando, esto es mediante el siguiente comando:
    - firebase login
4. esto abrira una pestaña de tu navegador predeterminado y ahi firebase te guiara en los pasos siguientes.
4.1 para cerrar la cuenta en caso de ser requerido:
    - firebase logout

5. Una vez inciada la sesion, el siguiente comando comenzara la configuracion de firebase donde
    mediante opciones en la consola pedira que confirmes la configuracion:
    - firebase init

6.una vez terminada la configuracion y se quisiera publicar el sitio, el
    comando seria:
    - firebase deploy
